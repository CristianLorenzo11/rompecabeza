¡Bienvenido al juego de rompecabezas!

Instrucciones:

  1. Selección de piezas: Para mover una pieza, haz clic en ella. Seleccionará y resaltará.
  2. Intercambio de piezas: Después de seleccionar una pieza, haz clic en otra para intercambiarlas. Esto permite reorganizar las piezas para resolver el rompecabezas.
  3. Inicio del juego: Haz clic en el botón "Comenzar" para iniciar el juego. Las piezas se mezclarán aleatoriamente para crear un nuevo rompecabezas.
  4. Mezcla de piezas: Cada vez que comiences el juego, las piezas se mezclarán en una configuración aleatoria, garantizando un nuevo desafío en cada partida.
